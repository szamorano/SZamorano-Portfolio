A factorial is the product of all whole numbers from 0 to our chosen number. Let's find ours by entering a number.

The result will display here:

        function factorial(n) {
            if (n == 1) {
                return 1;
            }
            return n * factorial(n - 1);
        }

        function calculate() {
            var input = Number(document.getElementById("test1").value);
            var result = factorial(input);
            document.getElementById("output").innerHTML = result;
        }

Fizzbuzz

Enter a number between 1 and 100 in each box below. For each multiple of the 1st number, "Fizz" will print. For each multiple of the 2nd number, "Buzz" will print. For each multiple of both numbers "FizzBuzz" will print.

The result will display here:

                                    function fizzBuzz() {
            var firstNumber = Number(document.getElementById("fizzbuzzTest1").value);
            var secondNumber = Number(document.getElementById("fizzbuzzTest2").value);
            var counter = "";
            for (var i = 1; i <= 100; i++) {
                if (i % firstNumber === 0 && i % secondNumber === 0) {
                    counter += "FizzBuzz ";
                } else if (i % firstNumber === 0) {
                    counter += "Fizz ";
                } else if (i % secondNumber === 0) {
                    counter += "Buzz ";
                } else {
                    (counter += i + ", ")
                }
                document.getElementById("fizzbuzzOutput").innerHTML = counter
            }
        }
